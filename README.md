# ❄️ 细雪之舞 · 刷课神器

**细雪之舞 · 刷课神器** 是一款专为在线课程平台打造的效率工具，助你高效完成学习任务。  
它界面美观，操作简洁，支持多项实用功能，真正做到省时省力、清爽不打扰。

---

## 🌟 插件主要功能

✅ **秒课功能**  
一键秒看当前视频，节省宝贵时间，轻松完成课程要求。

⚡ **倍速播放**  
支持原速、0.5倍速、1.5 倍速、2.0 倍速，甚至高达 **16 倍速** 播放。

🔓 **解除复制限制**  
解除网页禁止复制的限制，摆脱复制按钮无法使用的困扰。

📌 **自定义快捷网址**  
可添加常用学习网站，自由切换，一键直达。

🧊 **磨砂卡片 UI**  
界面采用磨砂质感，风格统一，贴近 **iOS 审美**，赏心悦目。
